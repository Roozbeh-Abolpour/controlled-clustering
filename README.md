# Controlled Clustering via Model Predictive Control

This repository implements a **control-theoretic framework for online clustering**, where a Model Predictive Control (MPC) layer regulates the dynamics of an incremental K-means algorithm.

The key idea is to **control the evolution of a learning algorithm externally**, by shaping its input data stream, rather than modifying its internal structure.

---

## 🔍 Core Idea

Incremental learning algorithms can be interpreted as nonlinear dynamical systems:

x_{t+1} = F(x_t, z_t)

where:
- x_t: internal state (e.g., cluster centers)
- z_t: incoming data sample

We introduce a control input:

z̃_t = z_t + u_t

where u_t is generated by a **Model Predictive Controller (MPC)**.

This transforms clustering into a **controlled dynamical system**, enabling:

- stabilization of cluster trajectories  
- reduction of oscillatory behavior  
- improved convergence properties  

without altering the underlying clustering algorithm.

---

## 🧠 Methodology

The framework combines:

- Incremental K-means → learning dynamics  
- ARX modeling + RLS → online system identification  
- Model Predictive Control (MPC) → control input computation  

At each time step:
1. The clustering system evolves with incoming data  
2. The system dynamics are estimated online  
3. MPC computes an optimal perturbation u_t  
4. The modified input z̃_t is fed into the algorithm

## ⚙️ Installation

Clone the repository:

git clone https://github.com/Roozbeh-Abolpour/controlled-clustering.git  
cd controlled-clustering  

Install dependencies:

pip install -r src/controlled_clustering/requirements.txt  

(Optional) create a virtual environment:

python -m venv venv  
source venv/bin/activate   # Linux / macOS  
venv\Scripts\activate      # Windows  

---

## ▶️ Running the Example

Run the main experiment:

python src/controlled_clustering/main.py  

This script compares:

- Uncontrolled clustering  
- MPC-controlled clustering  

and visualizes the evolution of clustering performance over time.

---

## ⚠️ Important Notes on Tuning

The performance of the controller depends critically on:

- ARX model orders (n_u, n_y)  
- prediction horizon N  
- number of MPC iterations per time step  
- input constraint bound (control magnitude)  

Proper tuning is essential for stability and performance.

## 🚀 Research Contribution

This work introduces a new perspective:

Machine learning algorithms can be controlled as dynamical systems.

Instead of modifying model parameters, we regulate how data enters the system.

This opens new directions in:
- controlled learning systems  
- adaptive data-driven control  
- stability analysis of learning algorithms
